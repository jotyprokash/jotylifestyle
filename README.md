**JotyLifestyle** is a premium e-commerce platform built with Laravel 8, designed to provide a seamless and sophisticated shopping experience. This project serves as a robust foundation for modern retail applications, featuring a clean architecture and a curated user interface.

## Core Features

- **Dynamic Catalog**: Browse products across various categories and sub-categories.
- **Seamless Shopping Cart**: Intuitive cart management for a frictionless user journey.
- **Secure Authentication**: Robust user login and registration system.
- **Order Management**: Comprehensive tracking of orders and shipping information.
- **Premium UI/UX**: A dark-themed, responsive design tailored for high-end retail.

## Technical Stack

- **Backend**: Laravel 8.x (PHP 8.1+)
- **Frontend**: Vue.js 2.x, Bootstrap 4, Laravel Mix
- **Database**: SQLite (Development) / MySQL (Production)
- **Tooling**: Composer, NPM
